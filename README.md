# speedtest.py

Замеряет скорость интернета: последовательно скачивает файл по указанному URL N раз и выводит скорость по каждому запросу и итоговую статистику.

Зависимостей нет — только стандартная библиотека Python 3.

## Использование

```
python speedtest.py <url> [--count N] [--delay СЕК]
```

| Аргумент | Описание | По умолчанию |
|---|---|---|
| `url` | URL файла для скачивания | — |
| `--count`, `-n` | Количество запросов | 10 |
| `--delay`, `-d` | Пауза между запросами (с) | 0 |

## Примеры

```cmd
python speedtest.py https://speedtest.selectel.ru/100MB
python speedtest.py https://speedtest.selectel.ru/1GB
python speedtest.py https://upload.wikimedia.org/wikipedia/commons/a/a7/Camponotus_flavomarginatus_ant.jpg --count 5 --delay 1
```

## Вывод

```
[ 1/10]    1.43 с    100.0 МБ    69.93 МБ/с
[ 2/10]    1.51 с    100.0 МБ    66.23 МБ/с
...

Средняя скорость : 67.84 МБ/с
Минимум          : 63.12 МБ/с
Максимум         : 71.55 МБ/с
```

## Примечания

- Если сервер возвращает 429 с заголовком `Retry-After`, скрипт автоматически ждёт и повторяет запрос.
- Для публичных серверов вроде Wikimedia рекомендуется `--delay 1`, чтобы не получить бан.
- Результат зависит от размера файла: мелкие файлы (~1 МБ) дают заниженные показатели из-за накладных расходов на соединение. Для честного замера используйте файлы от 10 МБ.
